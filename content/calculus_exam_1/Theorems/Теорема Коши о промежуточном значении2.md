Формулировка:
```spoiler-markdown
Пусть $f(x): [a, b] \to \mathbb{R}, f(a) < f(b), f(x)$ - непрерывна, тогда: $\forall{C \in (f(a), f(b))}~~ \exists{c \in (a, b)}~~ f(c) = C$
```

Доказательство (через пределы):
```spoiler-markdown
1 случай: $C = 0$ 
Рассмотрим $a_{n} \to -0, b_{n} \to +0$, тогда по непрерывности: $\lim_{n \to \infty} f(a_{n}) = \lim_{n \to \infty} f(b_{n}) = f(c)$
По [[Теорема о переходе к пределу в неравенстве и лемма об отделимости от нуля.|теореме о переходе к пределу в неравенстве]]:
$$\begin{cases}
\lim_{n \to \infty} f(a_{n}) \leq 0 \\
\lim_{n \to \infty} f(b_{n}) \geq 0
\end{cases} \Rightarrow 0 \leq f(c) \leq 0 \Rightarrow f(c) = 0$$
2 случай: $C \neq 0$
Рассмотрим $g(x) = f(x) - C$, $g(x)$ непрерывна.
По условию теоремы: $g(a) = f(a) - C < 0 \land g(b) = f(b) - C > 0$
По пункту 1: $\exists{c}~~ g(c) = 0 \Rightarrow f(c) - C = 0 \Rightarrow f(c) = C ~~~ \square$
```

Другое название - Теорема Больцано-Коши
