Формулировка:
```spoiler-markdown
$$\exists{\lim_{x \to a} f(x)}~~ \Leftrightarrow \forall{\varepsilon > 0}~~ \exists{\delta}~~ \forall{x', x''}~~
\begin{cases}
0 < |x'-a| < \delta \\
0 < |x''-a| < \delta
\end{cases} \Rightarrow |f(x') - f(x'')| < \varepsilon$$
```

Д-во:
```spoiler-markdown
$\large{\implies}$
Пусть $\lim_{x \to a} f(x) = A$, тогда:  $|f(x') - f(x'')| = |f(x') - A + A - f(x'')| \leq |f(x') - A| + |f(x'') - A| < \dfrac{\varepsilon}{2} + \dfrac{\varepsilon}{2} = \varepsilon$

$\large{\impliedby}$
Рассмотрим $\underbrace{ \{x_{n}\} }_{ \forall{\varepsilon > 0}~~ \exists{N(\varepsilon)}} \neq \{a\}:~~ x_{n} \to a \Rightarrow \underbrace{ |x_{n} - a| < \delta \land |x_{m} - a| < \delta }_{ \forall{n,m > N(\varepsilon)}}$
Тогда получаем, что $\forall{\varepsilon > 0}~~ \exists{N(\varepsilon)}~~ \forall{n,m > N(\varepsilon)}~~ |f(x_{n}) - f(x_{m})| < \varepsilon \Rightarrow \{f(x_{n})\}$ - [[Фундаментальная последовательность.|фундаментальная последовательность]] $\Rightarrow$ по [[Критерий Коши.|критерию Коши]] $f(x_{n}) \to A$
Пусть $y_{n} \to a$. Проведём аналогичные действия и получим: $f(y_{n}) \to B$
Рассмотрим $\{t_{n}\}:~~ x_{1} y_{1}, x_{2}, y_{2},\dots \to a$. Вновь проведём действия и получим: $f(t_{n}) \to C$
Но так как $f(t_{n}) \to C$, то и $\lim_{n \to \infty} f(x_{n}) = \lim_{x \to \infty} f(y_{n}) = A = B = C \Rightarrow$ по Гейне: $\exists{\lim_{x \to a} }~~ f(x)~~~\square$
```
