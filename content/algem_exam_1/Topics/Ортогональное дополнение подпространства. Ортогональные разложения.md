### Определение: Ортогональное дополнение
```spoiler-markdown
Пусть $S$ - подпространство в $V$. Множество всех векторов, ортогональных каждому вектору из $S$, называется **ортогональным дополнением** (ортодоп)
$$S^{\perp} = \{a \in V ~|~ \forall{x \in S}~~ a\perp x\}$$
```

### Предложение об ортогональном дополнении
Формулировка:
```spoiler-markdown
Пусть $S$ - подпространство пространства $V$ со скалярным произведением, а $S^{\perp}$ - ортогональное дополнение $S$. Тогда:
1) $S^{\perp}$ - подпространство $V$
2) Если $a_{1}, a_{2}, \dots, a_{k}$ - базис $S$, то $x \in S^{\perp} \iff xa_{1} = \dots = xa_{k} = 0$
```

Д-во:
```spoiler-markdown
Пункт 1.
Проверим $S^{\perp}$ на [[Подпространства. Действия с подпространствами#Подпространства|подпространство]], $x, y \in S^{\perp}, a \in S, \lambda \in F$:
$$(x+y)a = xa + ya = 0 + 0 = 0 \Rightarrow (x+y) \perp a \Rightarrow x+y \in S^{\perp}$$
$$(\lambda x)a = \lambda(xa) = \lambda \cdot 0 = 0 \Rightarrow \lambda x \perp a \Rightarrow \lambda x \in S^{\perp}$$

Пункт 2.
Если $a_{1}, a_{2}, \dots, a_{k}$ - базис $S$, а $x \in S^{\perp}$, то вектор $x$ ортогонален векторам $a_{1}, a_{2}, \dots, a_{k}$. Обратное, предположим, что $x \perp a_{1}, a_{2}, \dots, a_{k}$. Пусть $a \in S$, тогда:
$$\begin{align}
a &= \lambda_{1} a_{1} + \lambda_{2} a_{2} + \dots + \lambda_{k} a_{k} \\
ax &= (\lambda_{1} a_{1} + \lambda_{2} a_{2} + \dots + \lambda_{k} a_{k})x = \lambda_{1} (a_{1}x) + \lambda_{2} (a_{2}x) + \dots + \lambda_{k} (a_{k}x) = \\
&= \lambda_{1} \cdot 0 + \lambda_{2} \cdot 0 + \dots + \lambda_{k} \cdot 0 = 0
\end{align}$$
и потому $x \in S^{\perp}$ $~~~\square$
```

### Теорема об ортогональном разложении
Формулировка:
```spoiler-markdown
Если $V$ - пространство со скалярным произведением, а $S$ - подпространство в $V$, то $V = S \oplus S^{\perp}$ (**ортогональное разложение**)
```

Д-во:
```spoiler-markdown
Если $x \in S \cap S^{\perp}$, то $xx = 0 \Rightarrow x = 0$. А значит $S \cap S^{\perp} = \{0\}$, а значит $S + S^{\perp}$ - прямая. Осталось проверить, что $S + S^{\perp} = V$.

Положим $\mathrm{dim}~V = n, \mathrm{dim}~S = k$. Возьмём ортонормированный базис $a_{1}, a_{2}, \dots, a_{k}$ подпространства $S$ и дополним этот базис до ортонормированного базиса пространства $V$. Пусть $a_{k+1},\dots,a_{n}$ - вектора, использованные для дополнения. Так как каждый из этих $n-k$ векторов ортогонален $a_{1}, a_{2}, \dots, a_{k}$, по [[#Предложение об ортогональном дополнении|предложению]]: $a_{k+1}, \dots, a_{n} \in S^{\perp}$.

Итак, $S + S^{\perp}$ содержит все вектора $a_{1}, a_{2}, \dots, a_{n}$, составляющие базис $V$, откуда $S + S^{\perp} = V$ $\square$
```

### Свойства ортогонального дополнения
Формулировка:
```spoiler-markdown
Пусть $V$ - пространство со скалярным произведением, $S, S_{1}, S_{2}$ - его подпространства. Тогда:
1) $V^{\perp} = \{0\},~~ \{0\}^{\perp} = V$
2) $(S^{\perp})^{\perp} = S$
3) $S_{1} \subseteq S_{2} \Rightarrow S_{2}^{\perp} \subseteq S_{1}^{\perp}$
4) $(S_{1} + S_{2})^{\perp} = S_{1}^{\perp} \cap S_{2}^{\perp},~~ (S_{1} \cap S_{2})^{\perp} = S_{1}^{\perp} + S_{2}^{\perp}$
5) $V = S_{1} \oplus S_{2} \Rightarrow V = S_{1}^{\perp} \oplus S_{2}^{\perp}$
```

Д-во:
**Свойство 1.**
```spoiler-markdown
Если $x \in V^{\perp}$, то $\forall{y \in V}~~ xy = 0$, в частности $xx = 0$, а значит по [[Аксиомы евклидовых и унитарных пространства.#Скалярное произведение|4 аксиоме]] $x = 0$. Следовательно, $V^{\perp} = \{0\}$.
Равенство $\{0\}^{\perp} = V$ вытекает из [[Ортогональные системы. Процесс ортогонализации Грама–Шмидта. Ортонормированный базис#Замечания по определению|замечания о нулевом векторе и ортогональности]].
```

**Свойство 2.**
```spoiler-markdown
Из [[#Определение Ортогональное дополнение|определения ортогонального дополнения]] вытекает, что если $x \in S$, то $\forall{a \in S^{\perp}}~~ a\perp x$. Следовательно, $S \subseteq (S^{\perp})^{\perp}$. Пусть $\mathrm{dim}~S = k, \mathrm{dim}~V = n$. В силу [[#Теорема об ортогональном разложении|теоремы об ортогональном разложении]]:
$$\mathrm{dim}~(S^{\perp})^{\perp} = n - \mathrm{dim}~S^{\perp} = n - (n - k) = k = \mathrm{dim}~S$$
Итак, $S$ - подпространство $(S^{\perp})^{\perp}, \mathrm{dim}~S = \mathrm{dim}~(S^{\perp})^{\perp}$. Отсюда $S = (S^{\perp})^{\perp}$
```

**Свойство 3.**
```spoiler-markdown
Пусть $S_{1} \subseteq S_{2}, x \in S_{2}^{\perp}$. Тогда $\forall{a \in S_{2}}~~ x \perp a$, а значит и $\forall{a \in S_{1}}$. Следовательно, $x \in S^{\perp}_{1}$ и потому $S_{2}^{\perp} \subseteq S_{1}^{\perp}$
```

**Свойство 4.**
```spoiler-markdown
Пусть $x \in S_{1}^{\perp} \cap S_{2}^{\perp},~~ y \in S_{1} + S_{2}$. Тогда $y = y_{1} + y_{2},~ y_{1} \in S_{1}, y_{2} \in S_{2}$. В силу выбора $x$ имеем:
$$xy_{1} = xy_{2} = 0$$
$$xy = x(y_{1} + y_{2}) = xy_{1} + xy_{2} = 0 + 0 = 0$$
Значит $x \in (S_{1} + S_{2})^{\perp}$, потому $S_{1}^{\perp} \cap S_{2}^{\perp} \subseteq (S_{1} + S_{2})^{\perp}$.

Докажем обратное включение. Пусть $x \in (S_{1} + S_{2})^{\perp}$. Так как $S_{1} \subseteq S_{1} + S_{2}$ и $S_{2} \subseteq S_{1} + S_{2}$, из свойства 3 вытекает, что $x \in S_{1}^{\perp}$ и $x \in S_{2}^{\perp}$, то есть $x \in S_{1}^{\perp} \cap S_{2}^{\perp}$, а значит $(S_{1} + S_{2})^{\perp} \subseteq S_{1}^{\perp} \cap S_{2}^{\perp}$.

Мы проверили, что $(S_{1} + S_{2})^{\perp} = S_{1}^{\perp} \cap S_{2}^{\perp}$. Используя свойство 2, получаем:
$$S_{1}^{\perp} + S_{2}^{\perp} = ((S_{1}^{\perp} + S_{2}^{\perp})^{\perp})^{\perp} = ((S_{1}^{\perp})^{\perp} \cap (S_{2}^{\perp})^{\perp})^{\perp} = (S_{1} \cap S_{2})^{\perp}$$
```

**Свойство 5.**
```spoiler-markdown
$S_{1} \cap S_{2} = \{0\}$. Используя свойства 1 и 4, имеем:
$$S_{1}^{\perp} + S_{2}^{\perp} = (S_{1} \cap S_{2})^{\perp} = \{0\}^{\perp} = V$$
Так как $S_{1} + S_{2} = V$, а значит, используя 1 и 4, получаем:
$$S_{1}^{\perp} \cap S_{2}^{\perp} = (S_{1} + S_{2})^{\perp} = V^{\perp} = \{0\}$$
Значит $V = S_{1}^{\perp} \oplus S_{2}^{\perp}$ $~~~\square$
```
