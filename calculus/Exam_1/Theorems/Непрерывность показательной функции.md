Формулировка:
```spoiler-markdown
$a^{x}$ - непрерывна, то есть $\forall{\varepsilon > 0}~~ \exists{\delta(\varepsilon)}~~ \forall{x}~~ |x-x_{0}| < \delta \Rightarrow |a^{x} - a^{x_{0}}| < \varepsilon$
```

Д-во:
```spoiler-markdown
Рассмотрим $u < v$ такие, что $v - u < \dfrac{1}{2}$
По [[Принцип Архимеда и его следствие о плотности рациональных чисел.|принципу Архимеда]]: $\exists{p, q \in \mathbb{Q}}~~ p < u < v < q \land q - p < 2(v - u) < 1$ (то есть $p$ и $q$ находятся от $u$ и $v$ на расстоянии меньшем, чем половина расстояния между $u$ и $v$)
$$a^{v}-a^{u} \leq a^{q} - a^{p} = a^{p}(a^{q-p} - 1) \underbrace{ < }_{ \Delta } 2a^{p}(a-1)(q-p) < 2a^{u}(a-1)2(v-u) = 4a^{u}(a-1)(v-u)$$
$\Delta$: из [[Неравенство Бернулли|неравенства Бернулли]]

Возьмём $\delta = \dfrac{\varepsilon}{4a^{x_{0}}(a-1)}$, тогда:
$$|a^{x} - a^{x_{0}}| \leq 4a^{\min\{x,x_{0}\}}(a-1)|x-x_{0}| < \dfrac{4a^{\min\{x,x_{0}\}} (a-1) \varepsilon}{4a^{x_{0}}(a-1)} = \varepsilon \cdot \dfrac{a^{\min\{x,x_{0}\}}}{a^{x_{0}}} < \varepsilon ~~~\square$$
```
