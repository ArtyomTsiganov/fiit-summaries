Формулировка:
```spoiler-markdown
Пусть на плоскости задана произвольная система координат. Тогда всякая прямая на плоскости может быть задана некоторым уравнением вида: 
$$Ax + By + C = 0,~~~ A \neq 0 \lor B \neq 0$$
Обратно, любое уравнение
$$Ax + By + C = 0,~~~ A \neq 0 \lor B \neq 0$$
задаёт некоторую прямую.
```

Д-во (прямое утверждение):
```spoiler-markdown
Пусть на плоскости задана система координат с началом в $O$, $l$ - прямая, $M_{0}(x_{0},y_{0}) \in l, \vec{a} = (r, s)$ - направляющий вектор. Ясно, что эти данные однозначно определяют прямую.

Пусть $M(x,y)$ - точка плоскости, $\vec{r_{0}}$ - радиус-вектор $M_{0}$, $\vec{r}$ - радиус-вектор $M$
$M \in l \iff \vec{a} \parallel \overrightarrow{M_{0}M} \iff \overrightarrow{M_{0}M} = t\vec{a}$ для некоторого $t$. Так как $\vec{r} = \vec{r_{0}} + \overrightarrow{M_{0}M}$, тогда 
$M \in l \iff \vec{r} = \vec{r_{0}} + t\vec{a}$ (векторное уравнение прямой)
Тогда получаем:
$$\begin{cases}
x = x_{0} + rt \\
y = y_{0} + st
\end{cases}$$
(параметрические уравнения прямой)

Выразим $t$ из системы и приравняем, получим равенство:
$$\dfrac{x-x_{0}}{r}=\dfrac{y-y_{0}}{s}$$
(каноническое уравнение прямой)

Равенство эквивалентно: $s(x-x_{0}) - r(y-y_{0}) = 0 \Rightarrow sx - ry - sx_{0} + ry_{0} = 0$
Положим $A := s, B := -r, C := -sx_{0} + ry_{0}$, тогда уравнение имеет вид
$$Ax + By + C = 0,~~~ A \neq 0 \lor B \neq 0 ~~~~~~~~\square$$
```

Д-во (обратное утверждение):
```spoiler-markdown
Рассмотрим уравнение $Ax + By + C = 0,~~~ A \neq 0 \lor B \neq 0$
Пусть $(x_{0}, y_{0})$ - произвольное решение этого уравнения, $l$ прямая, проходящая через точку $M_{0}(x_{0}, y_{0})$ коллинеарно вектору $(-B, A)$. Докажем, что эта прямая задаётся уравнением ${} Ax + By = C = 0 {}$. Напишем каноническое уравнение прямой $l$:
$$\dfrac{x-x_{0}}{-B} = \dfrac{y-y_{0}}{A} ~~~~~~~~~(*)$$
Преобразуем, получим $Ax + By - Ax_{0} - By_{0} = 0$. Так как $(x_{0}, y_{0})$ - решение $Ax + By + C = 0 \Rightarrow -Ax_{0} - By_{0} = C$, а значит $(*)$ равносильно $Ax + By + C = 0 ~~~\square$
```
