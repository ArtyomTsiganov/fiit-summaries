### Определение: Базис трёхмерного пространства
```spoiler-markdown
Базис трёхмерного пространства - упорядоченная тройка некомпланарных векторов.
Базис из $\vec{a}, \vec{b}, \vec{c}$  обозначим как $(\vec{a}, \vec{b}, \vec{c})$
```

### Теорема о разложении вектора по базису в пространстве
Формулировка:
```spoiler-markdown
Пусть $(\vec{a}, \vec{b}, \vec{c})$ - базис пространства, $\vec{x}$ - произвольный вектор. Тогда существуют, и притом единственные, числа $t_{1}, t_{2}, t_{3}$ такие, что: $\vec{x} = t_{1}\vec{a} + t_{2}\vec{b} + t_{3}\vec{c}$
```

Д-во:
```spoiler-markdown
Существование:
Отложим $\vec{a}, \vec{b}, \vec{c}, \vec{x}$ от $O$ и обозначим концы полученных отрезков через $A, B, C, M$
${} \vec{a} \cancel{ || } \vec{b} \Rightarrow \exists{! \pi}~~ O, A, B \in \pi {}$. Спроектируем $M$ на $\pi$ параллельно прямой $OC$ и на прямую $OC$ параллельно $\pi$. Обозначим полученные точки через $M'$ и $C'$ и положим $\vec{x}' := \overrightarrow{OM'}, \vec{c} := \overrightarrow{OC'}$

По теореме о разложении вектора по базису на плоскости $\vec{x}' = t_{1}\vec{a} + t_{2}\vec{b}$ для некоторых $t_{1}, t_{2}$.
$\vec{c}' || \vec{c} \neq \vec{0} \Rightarrow \vec{c}' = t_{3}\vec{c}$ для некоторого $t_{3}$. Тогда $\vec{x} = \vec{x}' + \vec{c}' = t_{1}\vec{a} + t_{2}\vec{b} + t_{3}\vec{c}$

Единственность:
От противного: пусть $\vec{x} = s_{1}\vec{a} + s_{2}\vec{b} + s_{3}\vec{c}$, тогда:
$$(t_{1} - s_{1})\vec{a} + (t_{2} - s_{2})\vec{b} + (t_{3} - s_{3})\vec{c} = \vec{0}$$
Если $t_{1} - s_{1} \neq 0 \Rightarrow \vec{a} = - \dfrac{t_{2} - s_{2}}{t_{1} - s_{1}} \cdot \vec{b} - \dfrac{t_{3} - s_{3}}{t_{1}-s_{1}} \cdot \vec{c}_{0}$. Но тогда $\vec{a}, \vec{b}, \vec{c}$ - компланарны, противоречие. Аналогично противоречие возникает и для $s_{2}, s_{3} ~~~\square$
```

### Замечание о координатах суммы и произведения на число
Формулировка:
```spoiler-markdown
Пусть $\vec{x} = (x_{1}, x_{2}, x_{3}), \vec{y} = (y_{1}, y_{2}, y_{3})$ в базисе $(\vec{a}, \vec{b}, \vec{c})$, $t$ - произвольное число. Тогда:
$\vec{x} + \vec{y} = (x_{1} + y_{1}, x_{2} + y_{2}, x_{3}+y_{3}), t\vec{x} = (t x_{1}, t x_{2}, t x_{3})$
Аналогичное справедливо для векторов на плоскости
```

Д-во:
```spoiler-markdown
Через разложение по базису и свойства операций.
```
